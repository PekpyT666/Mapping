//============================================//
//--------- TILED SETUP FOR D'LIRIUM ---------//
//============================================//

1) Run "Set up the map editor.bat" file.

2) Once Tiled opens, click "Enable Extensions" in the top right corner.


//============================================//
//- Second and subsequent start of the Tiled -//
//============================================//

You can use the "Set up the map editor.bat" file to open Tiled everytime you need it.
Or you can open Tiled directly using "D'LIRIUM.tiled-project"
(but note - this file will be automatically updated by steam, so DON'T COPY or DON'T RENAME it).
Create a link to one of them on your desktop or wherever you like.


//---------- GENERAL MAPPING RULES ----------//

1.  Do not make infinity sized maps!
2.  Do not make maps with height more than 12000px!
3.  "Embed in map" flag must be enabled when you creating tiles!
4.  Always use type "Based on Tileset Image" when you creating tiles!
5.  Always set "depth" parameter for all your layers!
6.  All layers below player must have "depth" parameter set more than 0, but less than 16000!
7.  All layers above player must have "depth" parameret set less than level height (map height) * -1, but higher than -15984!
8.  All resources used on the map must be stored in the same directory as the map itself, in a folder with the same name as the map!
9.  Wall objects must be placed 8px upper than wall tiles!
10. Save and test your map in the "%LOCALAPPDATA%\Delirium\maps" (you can use "Place Your Maps Here.bat" file to open this folder)!
11. Do not use Group Layers (D'lirium support only Tile Layers, Object Layers and Image Layers)!



//===========================================//
//-- НАСТРОЙКА TILED ДЛЯ РАБОТЫ С D'LIRIUM --//
//===========================================//

1) Запустите "Set up the map editor.bat" file .

2) После того как Tiled откроется, нажмите на кнопку "Включить расширения" в правом верхнем углу.


//============================================//
//---- Втарой и последующие запуски Tiled ----//
//============================================//

Вы можете использовать файл "Set up the map editor.bat" для открытия Tiled каждый раз, когда это необходимо.
Или вы можете открывать Tiled напрямую, используя файл "D'LIRIUM.tiled-project"
(но учтите - этот файл будет автоматически обновляться самим steam, поэтому НЕ КОПИРУЙТЕ и НЕ ПЕРЕИМЕНОВЫВАЙТЕ его).
Создайте ссылку на один из них на рабочем столе или в любом другом удобном для вас месте.


//--------- ГЛАВНЫЕ ПРАВИЛА МАППИНГА ---------//

1.  Не создавай карты бесконечного размера!
2.  Не создавай карты высотой больше, чем 12000 пикселей!
3.  При создании набора тайлов флаг "Встроить в карту" должен быть включен!
4.  При создании набора тайлов всегда используй тип "Основано на изображении набора тайлов"!
5.  Все слои должны иметь установленный параметр "depth"!
6.  Все слои, отрисовывающиеся под игроком должны иметь значение параметра "depth" больше 0, но не больше чем 16000!
7.  Все слои, отрисовывающиеся над игроком должны иметь значение параметра "depth" ниже чем высота комнаты * -1, но не ниже чем -15984!
8.  Все ресурсы, используемые на карте должны храниться в той же директории, что и сама карта, в папке с тем же именем, что и карта!
9.  Объекты стен должны располагаться на 8 пикселей выше самих тайлов!
10. Сохраняй и тестируй свои карты в "%LOCALAPPDATA%\Delirium\maps" (вы можете использовать "Place Your Maps Here.bat" файл что бы открыть эту папку)!
11. Не используй Группы Слоёв (D'lirium работает только с Слоями Тайлов, Слоями Объектов и Слоями Изображений)
