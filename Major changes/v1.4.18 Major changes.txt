//---------- DEBUGGING TOOLS ----------//

Since steam added new recording feature, which default keybind is Ctrl+F11, it is now a bad practice to use this keybind in games.
So, here is how we changed our wrap debug tools keybinds:
Ctrl+F11 --> Ctrl+F10
Ctrl+F10 --> Ctrl+F9
Ctrl+F9  --> Ctrl+F8

//---------- ENTITIES ----------//

Entity Dialogue now have "e_disable_sequence"/"e_enable_sequence" properties.
This allow you to start/end sequence mode directly from first/last dialogue entity respectively instead of using additional two Entity Sequencer with three e_sequence* properties.
You can still use old Entity Sequencer way to start/end sequence mode, but please use new way for "dialogue cutscenes" instead when possible. It's more clean and simplier.

Entity Switch's "e_target_on"/"e_target_off" was been reworked to use "e_delay" like any other entities' target properties do.
If you by any chance had a switch that had both "e_target" and "e_target_on"/"e_target_off" while also using "e_delay" - split such switch in to two switches with the same "e_name" but with different "e_delay".
